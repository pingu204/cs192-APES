Added Features
  • Filter schedules depending on the following user preferences:
    1. Maximum Number of Classes per Day
    2. Preferred Class Days in a Week
    3. Total Distance Travelled in a Day
    4. Earliest and Latest Time
    5. Duration of Break/Vacant Periods

  • Fixes in generating schedules after clearing

  • Course timeslot, location, and venue keys now either "lec", "disc", or "lab"

  • Added redirects to log in when in guest mode
  • Redraw scraping changes
  • UI changes
